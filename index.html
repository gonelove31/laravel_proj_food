Order Food - Dự án Laravel

Đây là dự án đặt đồ ăn được phát triển bằng Laravel.

Các tính năng chính:

  • Đặt món ăn trực tuyến
  • Quản lý đơn hàng
  • Theo dõi trạng thái đơn hàng
  • Thanh toán trực tuyến
  • Quản lý người dùng

Cài đặt và sử dụng:

  1. Clone repository
  2. Cài đặt dependencies: composer install và npm install
  3. Cấu hình môi trường: cp .env.example .env và php artisan key:generate
  4. Chạy migrations: php artisan migrate
  5. Khởi động server: php artisan serve
